# 🚀 Cloud Cost Prediction & Idle Resource Detection Using Machine Learning

---

## 📌 Project Overview

Cloud infrastructure platforms dynamically allocate computing resources such as CPU, memory, storage, and network bandwidth. Inefficient provisioning and underutilized instances often lead to unnecessary operational expenditure.

This project presents a **Machine Learning-based analytical framework** that:

- Predicts monthly cloud infrastructure cost  
- Detects idle cloud resources  
- Estimates potential cost savings  
- Provides a scalable foundation for intelligent cloud optimization  

---

## ❗ Problem Statement

Inefficient cloud resource allocation and idle infrastructure generate unnecessary operational costs, requiring an intelligent system to predict billing expenditure and identify underutilized resources.

---

## 🎯 Objectives

- Develop a regression model to predict monthly cloud billing cost  
- Build a classification model to detect idle resources  
- Evaluate performance using statistical metrics  
- Estimate potential cost savings through optimization  

---

## 📂 Dataset Description

A structured cloud workload dataset containing **2000 records** was generated to simulate enterprise-level billing logs.

### 🔎 Features Included

- CPU Usage (%)  
- Memory Usage (%)  
- Network In / Network Out  
- Storage Usage (GB)  
- Uptime Hours (Monthly Cycle)  
- Instance Type  
- Region  
- Cost  
- Idle Status  

The dataset represents a **monthly billing cycle (up to 720 uptime hours)**.

---

## ⚙️ Methodology

### 1️⃣ Cost Prediction (Regression)

- Algorithm: **Random Forest Regressor**
- Train-Test Split: 80 / 20
- Hyperparameter Optimization Applied
- 5-Fold Cross Validation

#### 📊 Performance Metrics

- **R² Score:** 0.95 – 0.99  
- Low Mean Absolute Error (MAE)  
- Low Root Mean Squared Error (RMSE)  
- Stable cross-validation performance  

---

### 2️⃣ Idle Resource Detection (Classification)

- Algorithm: **Random Forest Classifier**
- Evaluated using:
  - Accuracy  
  - Confusion Matrix  
  - Precision  
  - Recall  

#### 📊 Performance

- **Accuracy:** > 97%  

---

## 💰 Business Impact

The system identifies idle cloud instances and calculates the total billing cost generated by underutilized resources, enabling:

- Cost optimization strategies  
- Infrastructure downsizing recommendations  
- Monthly savings estimation
